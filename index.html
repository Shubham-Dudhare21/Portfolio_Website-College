C#

projects

Versatile, object-oriented programming language developed by Microsoft as part of the .NET framework. C# is widely used for building desktop, web, and mobile applications, as well as game development with platforms like Unity. It offers features such as strong typing, garbage collection, and extensive standard libraries, making it suitable for a variety of development tasks. C# syntax is similar to other C-style languages, making it accessible to developers familiar with languages like C++ and Java.

C++

projects

Powerful, general-purpose programming language renowned for its performance, efficiency, and flexibility. C++ is commonly used in developing system software, games, desktop applications, and high-performance software. Its features include object-oriented programming, low-level memory manipulation, and a rich standard library. With its wide range of applications and strong support for multi-paradigm programming, C++ remains a popular choice for developers seeking high performance and fine-grained control over their code.

Python

projects

Versatile, high-level programming language known for its simplicity and readability. Python is widely used in web development, data analysis, artificial intelligence, scientific computing, and automation. Its extensive standard library and large ecosystem of third-party packages make it suitable for a wide range of applications. Python's clear syntax and dynamic typing facilitate rapid development and prototyping, making it a popular choice for both beginners and experienced developers.

Lua

projects

Lightweight, high-level scripting language designed for embedding in applications. Lua is known for its simplicity, efficiency, and versatility, making it ideal for use in video games, software development, and embedded systems. With a small footprint and easy integration, Lua is often used for scripting tasks, configuration files, and extending the functionality of larger applications.

Photoshop

Blender

projects

Open-source 3D creation suite offering a wide range of tools for modeling, animation, rendering, compositing, and motion tracking. Renowned for its versatility and community-driven development, Blender is used in various industries, including animation, visual effects, video game development, and architectural visualization. It supports a comprehensive set of features, including sculpting, rigging, simulation, and video editing, making it a powerful all-in-one solution for 3D content creation.

ZBrush

projects

Digital sculpting and painting software renowned for its powerful and intuitive tools for creating high-resolution 3D models. Utilized extensively in the fields of gaming, film, and digital art, ZBrush offers features such as dynamic tessellation, sculpting brushes, and customizable workflows. It allows artists to sculpt intricate details with ease, facilitating the creation of lifelike characters, creatures, and environments.
